关于 Anaconda

Anaconda 是一个免费开源的 Python 和 R 语言的分发版本，用于进行大规模数据处理、预测分析和科学计算，以满足开发者在大数据科学分析工作中需要使用的各种包。它的设计目标是简化包的管理和部署。Anaconda 的包使用软件包管理系统 Conda 进行管理。

Conda 作为包管理器可以帮助您查找和安装包。如果你需要一个需要不同版本Python的包，你不需要切换到不同的环境管理器，因为conda也是一个环境管理器。只需几个命令，您就可以设置一个完全独立的环境来运行不同版本的 Python，同时继续在正常环境中运行常用版本的 Python。

总的来说，Anaconda 是一个非常方便的工具，尤其是对于进行科学计算和数据分析的 Python 开发者来说。

安装 Anaconda

进入Anaconda官网进行下载安装包，选定安装路径后一路Next就成功安装了。

为了能够在cmd中控制Anaconda，我们需要为其添加路径。打开环境变量 - 在系统环境变量中新建ANACONDA_HOME，其值为Anaconda的安装路径。然后再系统变量的Path中添加三个路径：%ANACONDA_HOME%, %ANACONDA_HOME%\Scripts, %ANACONDA_HOME%\Library\bin。

然后您就成功安装并配置好Anaconda了，可以在cmd中输入conda -V来测试是否Anaconda在正常运行。
