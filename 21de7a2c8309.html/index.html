1.2 Experiment, Sample Space and Random Event

1.2.1 Basic Definitions

1. 随机试验（Random experiment，记作 $E$）

随机试验是在一致的环境下进行的试验，其结果却不可预知，只能依赖于随机的、不可预见的因素。例如，抛硬币、掷骰子和从一副牌中随机抽取一张牌，都属于随机试验。

2. 样本点或基本事件（Sample point or Elementary event，记作 $\omega$）

样本点或基本事件是随机试验可能产生的单一结果。比如，抛硬币的结果可能是“正面”或“反面”，掷骰子的结果可能是1到6中的任一数字。

3. 样本空间（Sample space，记作 $\Omega$）

样本空间是由随机试验所有可能的基本事件所构成的集合。例如，掷一个六面骰子的样本空间是{1，2，3，4，5，6}，而抛硬币的样本空间则包括{正面，反面}。

4. 随机事件（Random event，记作 $A, B, C…$）

随机事件是指在随机试验中可能发生的一个或一组结果，这些结果可以清晰地被定义和检测。一般来说，一个随机事件是样本空间的一个子集。例如，抛硬币的随机事件可以是“正面”或“反面”，掷骰子的随机事件可以是”得到偶数”，这个事件包含了样本空间的三个元素：2、4和6。

随机事件可以分类为：

  • 基本事件：如果一个事件仅包含样本空间的一个样本点，那么这就是一个基本事件。比如，掷骰子的结果为“4”。

  • 复合事件：如果一个事件包含了样本空间的多个样本点，那么这就是一个复合事件。比如，掷骰子的结果为偶数，这个事件包含了样本空间的三个样本点：2、4和6。

  • 对立事件：如果事件A的发生意味着事件B不会发生，反之亦然，那么A和B就是对立事件。例如，在抛硬币试验中，“正面”和“反面”就是一对对立事件。

  • 独立事件：如果事件A的发生与否并不影响事件B的发生概率，反之亦然，那么A和B就被认为是独立事件。例如，在连续抛两枚硬币的试验中，第一枚硬币的结果并不影响第二枚硬币的结果。

  • 互斥事件：如果两个事件不能同时发生，那么它们就是互斥事件。例如，掷骰子的结果为偶数和奇数是互斥的，因为一次掷骰子的结果不能同时为偶数和奇数。

2.3 Mathematical Expectation and Variance

Expectation:

定义：$\mu=E(X)=\sum\limits{i=1}^{+\infty}x{i}p(x{i})=\int{-\infty}^{+\infty}xf(x)dx$

嵌套：$E(g(X))=\sum\limits{i=1}^{+\infty}g(x{i})p(x{i})=\int{-\infty}^{+\infty}g(x)f(x)dx$

性质：$E(aX+b)=aE(x)+b$

Variance:

定义：$\sigma^{2}=Var(X)=E((X-\mu)^{2})=E(X^{2})-E(X)^{2}$

标准差：$\sigma=SD(X)=\sqrt{Var(X)}$

性质：$Var(aX+b)=a^{2}Var(X)$

Markov’s inequality: $P(X\geq\varepsilon)\leq\frac{\mu}{\varepsilon}$

Chebyshev’s inequality: $P(|X-\mu|\geq\varepsilon)\leq\frac{\sigma^{2}}{\varepsilon^{2}}$

2.4 Discrete Random Variables

1. Bernoulli Distribution:

$X\sim B(1, p)$ 进行一次试验，实验结果只有成功与失败，其中成功的概率为$p$。

  • $p(x) = \left{\begin{array}{ll}1-p & \text{if } x = 0 \p & \text{if } x = 1
    \end{array}\right.$

  • $E(X)=p$

  • $Var(X)=p(1-p)$

2. Binomial Distribution:

$X\sim B(n, p)$ 进行$n$次伯努利试验，其中伯努利成功的概率为$p$。

  • $p(x)=C_{n}^{p}p^{x}(1-p)^{n-x}$

  • $E(X)=np$

  • $Var(X)=np(1-p)$

3. Geometric Distribution:

$X\sim G(p)$ 持续进行伯努利试验，直到第一次成功，其中伯努利成功的概率为$p$。

  • $p(x)=(1-p)^{x-1}p$

  • $E(X)=\frac{1}{p}$

  • $Var(X)=\frac{1-p}{p^{2}}$
  • 无记忆性

4. Poisson Distribution:

$X\sim Poi(\lambda)$ 进行多次$p$很小的伯努利试验，其中$\lambda=np$。

  • $p(x)=\frac{\lambda^{x}}{x!}e^{-\lambda}$

  • $E(X)=Var(X)=\lambda$

2.5 Continuous Random Varibles

1. Uniform Distribution:

$X\sim U(a, b)$ 在区间[a, b]概率均匀分布。

  • $f(x) = \left{\begin{array}{ll}\frac{1}{b-a} & \text{if } a\leq x \leq b, \0 & \text{otherwise. }
    \end{array}\right.$

  • $E(X)=\frac{a+b}{2}$

  • $Var(X)=\frac{(b-a)^{2}}{12}$

2. Exponential Distribution:

$X\sim Exp(\lambda)$ 描述随着时间的变化。

  • $f(x) = \left{\begin{array}{ll}\lambda e^{-\lambda x} & \text{for } x > 0, \0 & \text{for }x \leq0.
    \end{array}\right.$

  • $F(x) = \left{\begin{array}{ll}1-e^{-\lambda x} & \text{for } x \geq 0, \0 & \text{for }x < 0.
    \end{array}\right.$

  • $E(X)=\frac{1}{\lambda}$

  • $Var(X)=\frac{1}{\lambda^{2}}$
  • 无记忆性

3. Normal Distribution:

$X \sim N(\mu, \sigma^{2})$ 正态分布。

  • $f(x)=\frac{1}{\sqrt{2\pi}\sigma}e^{\frac{-(x-\mu)^{2}}{2\sigma^{2}}}$

  • $\mu=0, \sigma^{2}=1$的正态分布为标准正态分布，标准正态分布的分布函数记为$\Phi(x)$

  • 使用$\frac{X-\mu}{\sigma}$进行标准化，然后使用给定的$\Phi(x)$进行计算

3.6 Numerical Characteristics of Random Vectors

  1. 随机变量求期望：$E(h(x, y))=\iint\limits_{R^{2}}h(x, y)f(x, y)dxdy$

    • 随机变量和的期望：$E(X+Y)=E(X)+E(Y)$
    • $X\bot Y$时，随机变量积的期望: $E(XY)=E(X)E(Y)$
  2. 随机变量求方差：$Var(X+Y)=Var(X)+Var(Y)+2Cov(X, Y)$
  3. 协方差：$Cov(X, Y)=E(XY)-E(X)E(Y)$，当$X\bot Y$时，$Cov(X, Y)=0$
    • $Cov(X, Y)=Cov(Y, X)$
    • $Cov(X, X)=Var(X)$
    • $Cov(aX, bY)=abCov(X, Y)$
    • $Cov(X+Y, Z)=Cov(X, Z)+Cov(Y, Z)$

  4. 相关系数：$\rho(x, y)=\frac{Cov(X, Y)}{\sigma{X}\sigma{Y}}=E(\frac{X-\mu{X}}{\sigma{X}}\frac{Y-\mu{Y}}{\sigma{Y}})$

    • 描述线性关系，$\rho>0$：线性正相关；$\rho<0$：线性负相关；$\rho=0$：无线性关系；
    • 标准化系数：$\rho(aX+b, cY+d)=\rho(X, Y)$；$|\rho|\leq1$
    • 正态分布如果没有线性关系，那么就独立。
